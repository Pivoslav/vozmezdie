# Shared experiment artifacts

- **`filtered_gt_json/<document_id>.json`** — Rows in LLM/agent shape (`section`, `entry_eng`, `entry_rus`, `content_category`, `framing`, `context`). Loaded automatically when `ground_truth.json_rows_dir` is set in `config/pipeline_config.experiment_redo.json`.

- **`FILTER_MANIFEST.json`** — Audit trail: HTML row counts, drops by category policy vs generic-neutral framing, final row counts.

- **`TAXONOMY_REFERENCE_FOR_ASSESSORS.md`** — Digest of **`config/Categories Explained.html`** **filtered by `config/experiment_redo_filter.json`** (allowed framings/categories for assessors). Ends with an explicit **Allowed framing labels for JSON output** list. Regenerated by `scripts/export_assessor_taxonomy_reference.py` and `prepare_experiment_redo.py`.

**Blind assessment rule:** Assessors must not open `filtered_gt_json/` until after their labels are committed.

Filter rules live in **`config/experiment_redo_filter.json`**. To rebuild shared files:

```bash
python scripts/prepare_experiment_redo.py --no-archive
```

(Omit `--no-archive` only when you intentionally want to archive `data/output` again.)
